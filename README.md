# 2048 游戏

一个使用 HTML、CSS 和 JavaScript 实现的经典 2048 游戏，支持 PC 端和移动端访问。

## 项目结构

```
2048game/
├── index.html    # 游戏主页面
├── server.js     # Node.js 本地服务器
├── server.py     # Python 本地服务器
└── server.bat    # Windows 启动脚本
```

## 游戏规则

### 目标
通过合并相同数字的方块，获得 2048 方块即可获胜。

### 初始状态
- 4x4 的网格棋盘
- 游戏开始时随机生成 3 个数字为 2 的方块

### 操作方式

**PC 端：**
- 使用键盘方向键（↑↓←→）控制所有方块移动
- 在游戏区域拖拽鼠标，根据拖拽方向移动所有方块

**移动端：**
- 在游戏区域滑动手指，根据滑动方向移动所有方块

### 移动规则
- 所有方块会沿着拖拽/滑动方向移动
- 方块移动时会堆叠到该方向的边缘
- 相同数字的方块相撞时会合并成一个方块
- 合并后的数字为两个方块数字之和

### 新方块生成
- 每次移动后会随机生成一个新方块（包括普通移动和合并）
- 新方块数字概率：
  - 40% 概率为 2
  - 30% 概率为 4
  - 30% 概率为 8

### 游戏结束
- 当无法移动任何方块且没有可合并的相邻方块时，游戏结束

## 运行方式

### 方式一：直接打开
使用浏览器直接打开 `index.html` 文件即可开始游戏。

### 方式二：启动本地服务器

**使用 Node.js（推荐）：**
```bash
node server.js
```

**使用 Python：**
```bash
python server.py
```

**使用 Windows 脚本：**
双击运行 `server.bat` 文件。

### 访问地址
- 本地访问：http://localhost:8000
- 局域网访问：http://{本机IP}:8000

## 技术栈
- HTML5
- CSS3（响应式布局）
- JavaScript（ES6+）
- Node.js（可选，用于本地服务器）

## 特点
- 响应式设计，支持 PC 端和移动端
- 使用 localStorage 保存最高分记录
- 流畅的动画效果
- 简洁的视觉设计

## 开发说明

### 核心函数
- `init()` - 初始化游戏
- `render()` - 渲染游戏界面
- `move(direction)` - 处理方块移动
- `slideLeft()` - 向左滑动逻辑
- `rotateGrid(grid)` - 旋转网格
- `addRandomTile(forceValue)` - 添加随机方块
- `checkGameOver()` - 检查游戏是否结束

### 事件处理
- 键盘方向键事件
- 鼠标拖拽事件
- 触摸滑动事件
